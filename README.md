# yandex-practicum-master-ML-pipelines
Данный проект выполнялся в рамках совместной магистратуры ТГУ и Яндекс Практикума. Реализация ML-решения, выбор метрик и моделей с использованием пайплайна.

В работе используются:
1) Метод k-ближайших соседей;
2) Логистическая регрессия;
3) Дерево решений;
4) Метод опорных векторов

**Цель**: изучение постоянных покупателей и разработка персональных предложений для увеличения покупательской активности.

**Задачи**:
1) Промаркировать уровень финансовой активности постоянных покупателей;
2) Разработать ML-модель, которая предскажет вероятность снижения покупательской активности клиента в следующие три месяца;
3) Используя данные модели, провести сегментацию покупателей.

**Этапы**:
1) Предобработка данных и EDA;
2) Корреляционный анализ;
3) Использование пайплайна для выбора ML-моделей и гиперпараметров;
4) Анализ важности признаков с помощью SHAP;
5) Сегментация покупателей.
